📖 项目介绍

OpenClaw 并非传统聊天式AI，而是一款开源、自托管的AI智能体执行网关，被国内开发者亲切称为“小龙虾”——它就像你的24小时不休息的执行管家，一边对接GPT-5.4、Gemini 3.1、Kimi等主流大模型，一边连接你的电脑、手机及各类工具，能直接拆解指令、执行操作，真正实现“AI从对话到落地”的跨越。

本仓库以“实战落地”为核心，系统化整理 OpenClaw 的学习笔记、配置教程、实战案例、插件开发、问题排查等内容，覆盖从新手入门到高阶优化的全流程，无论你是上班族、开发者、学生还是自由职业者，都能在这里找到适配自己的学习内容，快速用 OpenClaw 提升效率、解决实际问题。

🤝 贡献指南
OpenClaw 正处于高速发展期，本仓库也欢迎所有开发者参与共建，一起完善这份“OpenClaw 学习之路”：

1. Fork 本仓库；

2. 创建特性分支：git checkout -b feature/xxx（xxx 为你要添加的内容，如“新增电商场景案例”）；

3. 提交修改：git commit -m '添加 xxx 内容'；

4. 推送分支：git push origin feature/xxx；

5. 提交 Pull Request，描述清楚修改内容，等待审核合并。

贡献方向：新增实战案例、优化教程文档、修复问题、补充插件开发教程等，所有有价值的贡献都会被收录！
